# RDMA Network Validation

## Overview
This directory contains tools and scripts for validating RDMA network performance.

## Validation Categories

### Connectivity
- **Directory**: [connectivity/](connectivity/)
- **Tests**: RDMA connectivity validation
- **Checks**: Node-to-node RDMA reachability

### Bandwidth
- **Directory**: [bandwidth/](bandwidth/)
- **Tests**: RDMA bandwidth testing
- **Checks**: Throughput meets minimum requirements

### Latency/Stability
- **Directory**: [latency-stability/](latency-stability/)
- **Tests**: Latency and stability checks
- **Checks**: Low-latency, stable performance

## Usage
Run validation tests in the following order:
1. Connectivity validation
2. Bandwidth validation
3. Latency/Stability validation
